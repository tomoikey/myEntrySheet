### 1. 次世代のゲームプラットフォームとその未来
### 理由
理由は、世界の技術発達によりこれからのゲームの姿がどのように変化していくのかが興味あるからです。
ゲーム機自体の容量やオンライン技術が一般に発達していない数十年前のゲーム機は1人か横の人と一緒に遊ぶことが主流で、限られた環境下でエンジニアの方達は最大限楽しいものや今までにないものを作ろうとしていました。
しかし「ニンテンドーDS」のようにインターネット回線を利用して遠くの人達と遊べるようになったり、「Wii」のように人が実際に体を動かして遊べるゲーム機が登場し人々を驚かせ技術躍進は目まぐるしいものになりました。
また近年ではクレジットカードなどに利用されている「リップマンホログラム」という光学技術を応用した非接触型タッチパネルが登場し、空間上に浮かび上がったオブジェクトを操作できる時代がやってきました。
現代の新しい技術を応用して「こんなゲーム機を作れるのではないか」「そのゲーム機ができたらこんなゲームソフトが作れるのではないか」と構想を練る楽しさと、このテクノロジー時代の果てしない技術の海と共に自分も成長できるという幸せがあります。
私も次世代を担うことのできるプログラマーとして新しいことに挑戦する心を大切にし、誰よりも技術に喰らいついていく人材になります。
